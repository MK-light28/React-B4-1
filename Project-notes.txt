Redux toolkit: Diff Between Redux and Redux toolkit:
=================================================

Redux toolkit is mainly created to solve three major issue with Redux

Configuring a redux store is too complicated : Redux toolkit offer  configureStore function to
manage/simply the configuration

Needs to add lots of packages: Redux toolkit reduces the need to add multiple packages to build
large scale application
